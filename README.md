# Qichacha 企查查创投库数据获取自动化
## 自动化
企查查的防爬很厉害，所以只能是利用代码自动化一些复制粘贴的程序。   

极验（Geetest）还是得手动，要留意定时跳出的验证框。

## 数据准确度
数据中仍存在不及时、不准确、重复的信息。

## 代码
场景：获得**养老**行业的公司信息

    相关附件：  
    - 月度主营业务汇总_sample.xlsx
